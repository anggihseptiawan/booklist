# Booklist

This site shows a bunch books provided by Google books API.

## Tech Stack

- React
- TypeScript
- Tailwind
- React Router
- React-hot-toast
- SWR
- React-infinite-scroll-component

## How to run the project

### Install dependencies

```
# using pnpm
pnpm i
# using yarn
yarn
# using npm
npm i
```

### Run the project

```
# using pnpm
pnpm dev
# using yarn
yarn dev
# using npm
npm run dev
```
